# RunCircle

지역 기반 러닝 경쟁 앱의 첫 프로토타입입니다.

## 포함된 범위

- 1 km, 3 km, 5 km 반경 선택
- 현재 위치 확인 버튼
- Mapbox 지도, 확대/축소, 반경 맞춤
- 지도 위 주변 러너 핀과 정보창 표시
- 거리/아웃도어/위성 지도 스타일 전환
- 주변 러너 프로필, 아이콘, 거리, 페이스, 주간 거리 표시
- 주간 경쟁 요약

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

Mapbox public access token을 화면의 입력창에 넣으면 브라우저 `localStorage`에 저장하고 지도를 불러옵니다.

실제 서비스 단계에서는 토큰의 URL 제한과 예산 알림을 걸어 사용하세요.
